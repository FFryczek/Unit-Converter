# Unit Converter

A simple webpage to convert:

- Kilometers ↔ Miles  
- Celsius ↔ Fahrenheit  
- Kilograms ↔ Pounds  

---

## How to use

1. Enter a number in **one** of the two boxes.  
2. Leave the other box empty.  
3. Click **Convert**.  
4. See the result in an alert popup.

---

## Tech

- HTML  
- CSS  
- JavaScript  

---

## Notes

- Only fill one input box at a time.  
- Results show in alert messages.
